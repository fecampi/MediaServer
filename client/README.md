# Frontend React + MUI - Media Server HLS

## Como rodar o frontend

1. Instale as dependências:
   ```bash
   cd client
   npm install
   ```
2. Inicie o frontend:
   ```bash
   npm start
   ```

O frontend roda na porta 3000 por padrão (veja `.env`).

## Build para produção

```bash
npm run build
```

O build será servido automaticamente pelo backend Express.
